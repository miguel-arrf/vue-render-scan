# Changelog

## 0.1.0 - 2026-06-30

Initial public release.

- Added Vue 3 plugin for visualizing component redraws.
- Added Nuxt module via `vue-render-scan/nuxt`.
- Added runtime APIs: `scan`, `setOptions`, `getOptions`, `getStats`, `getReport`, and `onRender`.
- Added render detection tests for updates, filters, disabled mode, reports, and listeners.
